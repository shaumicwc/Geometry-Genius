Q&A about JavaScript

Question:1

What is DOM and what is the purpose of DOM?

Answer :

Definition: The full form of the DOM is “ Document Object Model. Its an Application Programming Interface which is Cross Platform and Language Independent. It modifies documents structure, style, content and represent HTML, XHTML, XML Document as a tree.


Purpose of DOM :


  1. DOM provides the access of a document like HTML, XHTML and XML
  2. DOM makes a platform to be language neutral.
  3. DOM provides the dynamic usage of the script
  4. DOM provides a way to update the document’s content, structure and style.
  5. DOM is used with the language and used to create the objects that can be used as a model to allow the interface to be developed easily
